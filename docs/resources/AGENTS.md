# Agents: Resources

Cross-cutting guides that apply across multiple advertising channels. These files support planning, measurement, compliance, and learning from examples.

## Files

| File | Topic |
|------|-------|
| `ANALYTICS.md` | Tracking and reporting ad performance |
| `BUDGETING.md` | Budget planning and allocation |
| `LEGAL.md` | Legal, regulatory, and ethical considerations |
| `EXAMPLES.md` | Case studies and successful campaigns |
| `FAQ.md` | Common questions and troubleshooting |

## Content focus

- **Actionable:** Readers should leave with concrete next steps or checklists.
- **Channel-agnostic:** Reference channels by name and link to `../channels/` guides instead of repeating channel setup.
- **Current:** Prefer linking to official platform help centers over stale procedural detail that changes often.
- **Accurate:** Legal and compliance content must be clearly framed as overview, not legal advice.

## When editing

- Link to channel guides when mentioning a specific medium: `[Social Media Advertising](../channels/SOCIAL.md)`.
- Link to ad-templates for creative help: `../ad-templates/TEMPLATES.md`.
- Keep infographics and image URLs valid; flag placeholder URLs (e.g. `example.com`) if you see them.

## When adding a new resource

1. Add the file here, not under `channels/`, unless the topic is truly single-channel.
2. Update [../README.md](../README.md), [../../README.md](../../README.md), and [../meta/REPO_OUTLINE.md](../meta/REPO_OUTLINE.md).
3. Run `python scripts/check-internal-links.py`.
